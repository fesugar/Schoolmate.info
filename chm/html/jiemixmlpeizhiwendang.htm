操作实例：解密XML配置文件

本操作实例描述如何解密XML配置文件。

重要事项：

软件基于 .NET 平台编写，需要 .NET 4.0 的运行环境支持。如果未安装对应的运行环境时可能会出现 “0xc0000135” 的错误提示。。

先决条件

若要完成此操作实例，需要具备以下条件：

  • 保证 Microsoft .NET Framework 4 框架已经成功安装。

  • 修改 Schoolmate.Database.Conn.dll 为 Schoolmate.Database.Conn.exe

解密XML配置文件命令行选项

解密xml配置文件语法是：

Schoolmate.Database.Conn.exe < xmlfile > < model [ Encrypt or Decrypt ] >

命令解释：

  • xmlfile 命令。需要加解密的xml文件。

  • model 命令。加解密开关。（ Encrypt 加密 ） （ Decrypt 解密 ）

下列示例对 XML 配置文件进行解密：

Schoolmate.Database.Conn.exe C:\Schoolmate.info\Schoolmate.conn.xml Decrypt

 

下表描述了如何使用命令选项。这些选项区分大小写。

选项 参数 说明

Encrypt

对XML配置文件进行加密。

示例：

Schoolmate.Database.Conn C:\Schoolmate.info\Schoolmate.conn.xml Encrypt

Decrypt

对XML配置文件进行解密。

示例：

Schoolmate.Database.Conn C:\Schoolmate.info\Schoolmate.conn.xml Decrypt
