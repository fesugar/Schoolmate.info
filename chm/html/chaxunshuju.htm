操作实例：查询数据

本操作实例描述如何查询数据。

重要事项：

软件基于 .NET 平台编写，需要 .NET 4.0 的运行环境支持。如果未安装对应的运行环境时可能会出现 “0xc0000135” 的错误提示。。

先决条件

若要完成此操作实例，需要具备以下条件：

  • 保证 Microsoft .NET Framework 4 框架已经成功安装。

  • 保证所获取的软件包含全部文件，并且未作任何修改。

主界面

点击展开菜单栏。

展开菜单项

点击"查询"按钮。

查询数据界面

填写好要查询的数据后点击"查询"按钮进行数据的查询。
